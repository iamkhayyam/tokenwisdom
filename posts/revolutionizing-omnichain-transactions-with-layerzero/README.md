How does LayerZero work?

3 contracts. 2 chains. 1 transaction.

//LFG as the kids say!!

LayerZero relies on smart contracts to process transactions quickly and securely. It uses messaging as the value proposition, making it a cost-effective alternative to traditional methods. LayerZero enables users to swap one blockchain’s native asset for another, known as a native swap. This means that a user can start with Ethereum’s native asset and end with Binance Smart Chain’s (BSC) native asset or Avalanche’s (AVAX) native asset, all in one transaction.

LayerZero’s multi-hop processing system executes the swap across different contracts, making it possible to transfer assets across any two arbitrary contracts across any blockchain. For another example, if a user wants to swap their X tokens for Y tokens, LayerZero will automatically find the best price and execute the trade across different contracts until the tokens are transferred to the user’s Y tokens wallet.
